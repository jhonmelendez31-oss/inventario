# inventario
# 🚗 Control Vehicular - Rondero

Sistema web para control e inventario vehicular.

Permite buscar placas, registrar vehículos, editar información,
eliminar registros, exportar e importar respaldos en JSON.

---

# ✨ Funciones

- 🔍 Búsqueda rápida por placa
- 📋 Generación de planillas
- ➕ Agregar vehículos
- ✏️ Editar vehículos
- 🗑️ Eliminar vehículos
- 💾 Exportar respaldo JSON
- 📂 Importar respaldo JSON
- 🎨 Colores dinámicos
- 📱 Diseño responsive
- ⚡ Sistema rápido y ligero
